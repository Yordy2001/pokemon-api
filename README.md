# Pokemon-api

This is a Pokemon api done by node(express) to practice backend





## Tech Stack


**Server:** Node, Express, postgres, sequelize


## Run Locally

Clone the project

```bash
  git clone https://github.com/Yordy2001/pokemon-api.git
```

Go to the project directory

```bash
  cd pokemon-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
